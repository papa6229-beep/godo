고도몰 상세페이지 생성기 가이드

1. 기존 생성기의 모든 기능적 부분은 그대로 따라간다
- 좌측 입력영역의 UX, UI도 그대로 따라간다
- 기본설정, 스펙정보, ai생성 참고용 핵심요약, 이미지 등을 참고하여 ai가 문구생성가능
- ai의 문구생성가능 영역은 선택, 사람이 직접 입력 가능
- 각 섹션의 활성화, 비활성화 부분
- 메인이미지를 바탕으로 한 섬네일 생성
- 옵션, 패키지 이미지 생성 시 크기 위치 변경 가능
- 스펙정보의 무게를 입력하면 사이즈 섹션에 자동 반영
- 파일 저장방식
- 워터마크기능
- 임시저장, 불러오기 기능 등등

2. 이 부분만 기존 생성기의 기능에서 비활성화한다
- 동영상 삽입 (아예 삭제하지 말고 비활성화 / 나중에 살릴지도 모르니)

3. 이 부분은 기존 생성기에서 변경한다
- a.i 생성 참고용 핵심요약
  : 현재 하나의 텍스트 입력 영역으로 되어있는것을
  메인특징1(직접입력 필수) 
  특징설명 (ai가 문구 생성 가능)

  메인특징2(직접입력 필수) 
  특징설명 (ai가 문구 생성 가능)
 
  메인특징3(직접입력 필수) 
  특징설명 (ai가 문구 생성 가능)

  이렇게 3개의 입력영역으로 구분하며
  사용자가 입력하는 메인특징 3개는 a.i가 문구를 생성함에 있어
  핵심적으로 참고한다

  이 부분의 디자인적 변경점은 가이드 이미지와 예제 이미지를 참조


설명전 참고사항
- 설명의 편의 및 이해를 돕기위해 
  가이드 이미지와 이 가이드파일에
  생성기 좌측 입력영역의 각 대표 섹션에 번호를 부여함(위에서 아래로)

- 모든 방식은 그대로이며 고도몰 생성기에서는
   위치, 즉 레이아웃만 변경된다.

  1. 기본영역 (컬러테마, 상품명(한글), 영문 상품명
  2. 메인이미지영역
  3. 스펙정보 (특징, 타입/종류, 재질/소재 등등)
  4. 패키지 이미지 설정
  5. 추가 옵션
  6. 상세포인트 (Feature, 각영역 이미지, ai문구삽입영역, point 01, point02...)
  7. size

- 이 md파일과 설계도인 가이드.jpg 그리고 실제로 만들어졌을때를 상정한 예제.jpg를 같이 참조한다.
- 모든 폰트는 pretendard이며 가장 굵은폰트는 black, 중간은 bold, 일반 설명은 medium
- 3번의 제조사는 메인이미지 위에 겹쳐서 배치된다 (위치는 가이드 및 예제 이미지 참조)
- 패키지 이미지 영역도 메인이미지 위에 겹쳐서 배치된다 (위치는 가이드 및 예제 이미지 참조)
- 1번의 영문상품명은 2곳에 위치한다. (위치와 크기는 가이드 및 예제 이미지 참조)
- 각 영역 폰트의 굵기, 크기, 정렬, 섹션간 간격, 라인 생성등 레이아웃은 예제 이미지를 참조한다
   (가이드 이미지는 순수 설명서임)
